# Моя консольная канбан-доска
### Описание
**Канбан-доска** --- это такой вид управления задачами, в котором мы можем отслеживать статус работ над задачами через перемещения между столбцами доски.

По умолчанию у нас будет 3 столбца:
1. **ToDo** - задачи, которые мы хотим когда-либо сделать.
2. **In Progress** - задачи, которые мы делаем в данный момент.
3. **Done** - задачи, которые уже выполнены.

### Что в себе хранит задача?
Внутри задачи есть следующая информация:
- Название задачи - краткое описание того, что необходимо выполнить.
- Описание задачи - полное описание того, что нужно выполнить.
- Номер задачи - порядковый номер задачи.
- Статус задачи - показатель того, в каком состоянии находится задача.

### Что можно сделать с задачами?
Задачу можно:
- Создать.
- Отредактировать.
- Удалить.
- Поменять статус.
- Получить описание задачи.
